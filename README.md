# Initialize Golang Project

This is a simple golang project that uses gin to create a server with endpoints. I am building on this to create a student sign up portal to manage the IFTP improv sign-ups and class rosters. 

To initialize a go project, run:
```
mkdir IFTP && cd IFTP
go mod init IFTP
```

Now we can make a main.go file:
```
touch main.go
```

And finally a user package with our user methods.
